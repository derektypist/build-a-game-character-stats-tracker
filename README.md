# Build a Game Character Stats Tracker

Allows creation of a character with specific attributes, update those attributes, and retrieve the current stats of the character.

## Technologies

Uses Python3.  Object-Oriented Programming with getters and setters are applied.

## Deployment

Copy `main.py` into the text editor of your choice.  If using terminal, use the command `python3 main.py` to run it.
